# รายการตรวจสอบ คำแนะนำ / สิ่งที่ต้องทำ ผ่าน ✓ ไม่ผ่าน ✗
1. ทางเดินสะอาด ไม่มีใบไม้/เศษขยะกีดขวาง กวาดทุกรอบ ตรวจทางเดินหลักและทางรอง
2. ตัดหญ้าสนามทั้งหมด ไม่ให้ยาวเกินไป ใช้เครื่องตัดหญ้า
3. ไม่มีน้ำขังหรือกลิ่นเหม็นเน่าในสวน ตรวจรางระบายน้ำ เปิดทางน้ำไหล
4. รดน้ำทั่วบริเวณพื้นหญ้า รดน้ำวันละ 1–2 ครั้ง ตามสภาพอากาศ
5. ท่อน้ำและระบบรดน้ำใช้งานได้ ไม่รั่ว เปิดทดสอบทุกรอบ พบรั่ว → แจ้ง + บันทึกชำรุด
6. โคมไฟสวน / หลอดไฟทางเดิน ติดครบ แจ้งช่างเปลี่ยนทันทีเมื่อพบดับ
7. ไม่มีแมลงศัตรูพืช / รังมด / รังแมลงในสวน กำจัดทันทีที่พบ
🌳 ต้นไม้และพุ่มไม้ (ตรวจทุกต้น)
# รายการตรวจสอบ คำแนะนำ / สิ่งที่ต้องทำ ผ่าน ✓ ไม่ผ่าน ✗
8. ต้นไม้/พุ่มไม้ ไม่มีกิ่งห้อยต่ำกีดขวาง ตัดกิ่งที่ห้อยต่ำกว่า 2 เมตร
9. ต้นไม้ได้รับน้ำครบ ดินไม่แห้งแตก รดน้ำตามกำหนด ตรวจความชื้นดิน
10. ใบไม้สีเขียวสดใส ไม่เหลืองหรือร่วงผิดปกติ หากพบเหลืองผิดปกติ หาวิธีจัดการทันที
11. พุ่มไม้ทรงพุ่มสวยงาม ไม่รกทะมึน ตัดแต่งตามรูปทรงที่กำหนด
12. ระบบน้ำหยด / สปริงเกอร์ ทำงานได้ปกติ พบชำรุด → แจ้ง + บันทึกชำรุดทันที
13. กระถาง / แปลงดอกไม้ สะอาด ไม่มีเศษขยะ เก็บเศษใบและขยะออกจากกระถางทุกรอบ
💨 ปั้มลม
# รายการตรวจสอบ คำแนะนำ / สิ่งที่ต้องทำ ผ่าน ✓ ไม่ผ่าน ✗
14. พื้นที่ปั้มลมสะอาดไม่มีขยะ เก็บเศษใบและขยะออกจากพื้นที่ทุกรอบ
15. เครื่องปั้มลมพร้อมใช้งาน ตรวจช่วงเช้า / และก่อนเลิกงาน
16. หน้าจอแสดงผลปกติ ตรวจช่วงเช้า / และก่อนเลิกงาน
17. หัวเติมลม สายลมไม่แตก ไม่ร้าว ไม่ชำรุด ตรวจช่วงเช้า / และก่อนเลิกงาน
🔆 ดูแลความเรียบร้อยทั้งหมด (15:00 น.)
# รายการตรวจสอบ คำแนะนำ / สิ่งที่ต้องทำ ผ่าน ✓ ไม่ผ่าน ✗
18. ใส่ปุ๋ยต้นไม้ตามแผน ใส่ปุ๋ยตามปริมาณที่กำหนด
19. พ่นยาฆ่าแมลงบริเวณที่พบการระบาด สวมอุปกรณ์ป้องกัน ติดป้ายเตือนหลังพ่น
20. ล้างทำความสะอาดทางเดินและลานสวนทั้งหมด ใช้สายยางล้าง ไล่น้ำเข้ารางระบาย
21. ตรวจและล้างท่อระบายน้ำในสวนทั้งหมด เปิดฝาท่อ เอาเศษใบออก ตรวจน้ำระบายได้ดี
22. รดน้ำต้นไม้ทั้งหมด (รอบใหญ่) รดให้ชุ่มถึงราก ตรวจน้ำไหลทั่วถึง
23. ตรวจสภาพรั้วและทางเดิน พบชำรุดบันทึก ตรวจรั้ว ประตู ไฟ รางน้ำ รายงานทันที
24. เก็บขยะและใบไม้ในสวนทั้งหมดใส่ถุง วางถุงขยะที่จุดรับขยะ ไม่ทิ้งค้างในสวน
25. ล้างและจัดเก็บอุปกรณ์สวนให้เรียบร้อย ล้างกรรไกร จอบ สายยาง เก็บเข้าห้องอุปกรณ์
